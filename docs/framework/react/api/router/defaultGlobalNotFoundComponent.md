# DefaultGlobalNotFound 컴포넌트

`DefaultGlobalNotFound` 컴포넌트는 다른 매칭되는 라우트가 없고 `notFoundComponent`가 제공되지 않을 때, 루트 라우트에서 "Not Found"를 렌더링하는 컴포넌트입니다.


## DefaultGlobalNotFound 반환값

```tsx
Not Found
```
